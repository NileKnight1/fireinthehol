## 2d game made by godot engine for hackclub :)

- The game idea is so simple, you have to survive the fire flying at you. 
- The hardness increase each level, and you get coins accordingly.
- Coins has no use yet, but will have soon.

## Gameplay
- Use A and D to move left and right, shift to sprint, space to jump and E to interact.
- Mobile controls are in development.
